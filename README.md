## About
**KRMI-Collection** is a repository intended to store and organize KRMI-related content in one place, such as:
- documentation / notes
- code utilities and scripts
- references and links
- datasets or examples (if applicable)

### Goals
- Centralize KRMI resources
- Keep a consistent structure
- Make it easy to discover, reuse, and contribute improvements
